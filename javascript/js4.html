JavaScript - Lesson 3

This lesson will go over loops, arrays, variable declaration keywords, if statements, and functions/methods.

A note: capitalization is very important in JS. Keywords like var and while must have the proper capitalization, which is no caps in this case. If your code is giving you errors, make sure you have the right capitalization.

There are some restrictions on what you can name your variables as well, but common practice is to use camel casing, or capitalizing the beginning of every word except the first, like this: exampleVariableName. Another thing to note is that the first character of a variable name cannot be a number.
